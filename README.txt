Лабораторна робота №1 (Варіант 11)

Скласти блок-схему алгоритму та програму мовою Python згідно з варіантом,
отриманим від викладача, виконати відповідне завдання за табл. 1.
Відповідний алгоритм оформити у вигляді функції; 
програма повинна вивести на екран отриману матрицю.
NB: під час розроблення програми НЕ МОЖНА: 
використовувати вбудовані в Python функції, 
які б могли вирішити завдання в один прохід.  
Наприклад, Напишіть функцію для перевертання рядка -> НЕ можна використати 
вбудовану функцію reverse для перевертання рядка. 
Її потрібно реалізувати самостійно
